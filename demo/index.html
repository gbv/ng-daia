daia-simple

ng-daia includes a filter, a template (template/daia-simple.html), and a directive to handle DAIA Simple format.

code

<div daia-api="{{myAPI}}" 
     daia-id="{{myID}}" 
     daia-filter="daiaSimple"></div>
